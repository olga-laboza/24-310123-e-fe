
1 уровень сложности: 

	1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?

	Предварительно, до выполнения команды копирования необходимо авторизоваться на gitHub и загрузить SSH ключ.
	Его наличие можно проверить, зайдя в настройки профиля Settings ?SSH and GPS keys. 
	Также на локальном компьютере необходимо создать директорию, в которую мы будем копировать удаленный репозиторий. 
	Затем в командной строке gitBash перейти в созданную директорию и  выполнить команду git clone <url>, 
	где <url> - адрес удаленного репозитория, который необходимо скопировать, зайдя на github.com в необходимый репозиторий,
	нажать на кнопку «Code» и выбрать вкладку «SSH».
	

	2. Как получить на локальный компьютер изменения с github.com

	Для получения изменений с GitHub используется команда git pull, которую необходимо ввести в командной строке gitBash, 
	находясь в соответствующей директории.


	3. Какую информацию нам дают команда : git status git log git remote -v 

	Команда git status – отображает текущее состояние staging area (информацию об объектах, подготовленных для коммита);
	git log – отображает информацию (историю) о совершенных коммитах;
	git remote -v – показывает, с каким удаленным репозиторием связан наш локальный репозиторий и отображает адреса для чтения и записи,
	привязанные к настроенному удаленному репозиторию.
